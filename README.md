<h1>Демонстративная API</h1>
<p>
    Скачать:

```shell
$ git clone https://github.com/2tmirleid/impulse-docker-lesson.git
```

Запустить вне докера:

```shell
Режим разработчика:
$ npm run start:dev

Продакшн режим:
$ npm run start:prod
```

Запустить в докер контейнере:

```shell
Сборка:
$ docker-compose build

Запуск:
$ docker-compose up

ИЛИ
$ docker-compose up --build

Запуск на фоне:
$ docker-compose up -d
```
</p>
